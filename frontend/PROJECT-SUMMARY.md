## 🔐 Data Management

### What Gets Stored
```javascript
localStorage.cart          // Shopping cart items
localStorage.userProfile   // User information
localStorage.userAddresses // Delivery addresses
localStorage.userPassword  // Password (demo only)
localStorage.userPreferences // Notification settings
```

### Data Structure
```javascript
// Medicine object
{
  id, name, company, category, emoji,
  price, stock, batch, expiryDate, description
}

// Order object
{
  id, items[], totalAmount, status,
  date, timeline[]
}

// Cart item
{
  ...medicine, quantity
}
```

## 🔧 Customization Examples

### Change Brand Color
Edit `styles.css`:
```css
--primary-color: #FF0000; /* Your color */
```

### Add Medicine
Edit `app.js`:
```javascript
medicinesDatabase.push({
  id: 13,
  name: "New Medicine",
  // ... other fields
});
```

### Change Tax Rate
Edit `app.js`:
```javascript
getTax() {
  return this.getTotal() * 0.10; // 10% instead of 5%
}
```

### Add Category
1. Edit `index.html` select options
2. Add medicine category in `app.js`
